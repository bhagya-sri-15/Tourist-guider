# Tourist Guider Website 🌍✈️

## Overview
Tourist Guider Website is a simple and attractive travel guide website built using HTML, CSS, and JavaScript. It helps users discover popular tourist places and get travel-related information.

## Features
- Responsive Navigation Bar
- Home Section with Welcome Banner
- About Section
- Tourist Destinations Gallery
- Contact Form
- Smooth Scrolling
- Mobile Friendly Design
- Interactive JavaScript Effects

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
Tourist-Guider/
│── index.html
│── style.css
│── script.js
│── images/

## How to Run
1. Download the project files.
2. Open `index.html` in any browser.
3. Explore the website.
